MATE X
Diseñada para romper la rutina, lista para cualquier terreno
EXPERIMENTÁ LA LIBERTAD
La MATE X fue creada para quienes no siguen caminos, los inventan.
Con ruedas anchas y diseño robusto, es ideal para calles, montaña, ripio y viñedos.
En Mendoza o donde sea, cada viaje se convierte en una historia.
SMART LCD COLOR DISPLAY
Pantalla LCD a color
La pantalla inteligente muestra velocidad, distancia y nivel de asistencia. Además, permite cargar dispositivos por USB.
FAT TIRE
Neumáticos de 4 pulgadas
Los neumáticos de 20x4" brindan tracción y estabilidad, ideales para ciudad, tierra o arena.
SUSPENSION
Suspensión delantera y trasera
Equipado con doble suspensión para mayor confort y adaptabilidad en cualquier terreno.
BRAKING SYSTEM
Sistema de frenos
MATE.BIKE incorpora frenos a disco para máxima seguridad, disponibles en versiones mecánicas o hidráulicas.
Batería integrada en el cuadro
Todos los modelos MATE X vienen equipados con baterías de alto rendimiento de marcas líderes como Samsung, LG y BAK.
Dependiendo del lote, la batería puede variar, pero siempre se garantiza un estándar de calidad igual o superior al original.
¿La cargás? Como vos quieras: directamente desde la bici o sacando la batería para mayor comodidad.
